# 个人生活管理平台 - 前端项目

## 项目介绍
这是一个基于Vue 3和Element Plus构建的个人生活管理平台前端项目，包含日程管理、记账管理、备忘录、文件存储和个人中心等功能模块。

## 技术栈
- Vue 3
- Element Plus
- Vue Router
- ECharts (数据可视化)
- WangEditor (富文本编辑器)
- Vite (构建工具)

## 功能模块

### 1. 日程管理
- 添加/修改/删除日程
- 设置提醒功能
- 日程状态管理

### 2. 记账管理
- 记录收入/支出
- 月度统计图表
- 分类统计展示

### 3. 备忘录
- 写笔记（支持图片插入和格式化）
- 重要笔记置顶功能
- 笔记编辑和删除

### 4. 文件存储
- 上传图片/PDF文件
- 文件下载和预览
- 文件管理

### 5. 个人中心
- 修改密码
- 更换头像
- 导出数据备份

## 项目特色

### 设计风格
- 统一的主题色系统（渐变蓝紫色）
- 现代化的卡片式布局
- 精致的阴影和圆角设计
- 清晰的视觉层次结构

### 用户体验
- 流畅的页面过渡动画
- 丰富的交互反馈效果
- 响应式设计，支持移动端
- 直观的数据展示方式

## 安装和运行

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 构建生产版本
```bash
npm run build
```

### 4. 预览生产版本
```bash
npm run preview
```

## 注意事项

1. 请确保已安装Node.js 18+和npm 9+
2. 如果在TraeAI终端中遇到命令无法识别的问题，请尝试在系统自带的命令提示符(cmd)或PowerShell中运行
3. 开发服务器默认运行在 http://localhost:3000

## 项目结构

```
src/
├── views/            # 页面组件
│   ├── account/      # 记账管理
│   ├── file/         # 文件存储
│   ├── note/         # 备忘录
│   ├── profile/      # 个人中心
│   ├── schedule/     # 日程管理
│   ├── Home.vue      # 主页面
│   └── Login.vue     # 登录页面
├── router/           # 路由配置
├── App.vue           # 根组件
└── main.js           # 入口文件
```
